PY-AI ASSISTANT - SETUP GUIDE

1. INSTALL LIBRARIES (VERY IMPORTANT) Open your Command Prompt (CMD) or Terminal and run these two commands one by one:

pip install huggingface_hub

pip install customtkinter

2. GET YOUR API TOKEN

Create a free account at huggingface.co

Go to Settings -> Access Tokens

Create a new "Read" token and copy it.

3. EDIT THE CODE

Open main.py

Find: API_TOKEN = "YOUR_TOKEN_HERE"

Replace "YOUR_TOKEN_HERE" with your own token.

4. RUN

Run main.py and start chatting!
